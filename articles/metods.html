Функции (методы) в Java

В Java функции называются методами и всегда объявляются внутри классов.

1. Синтаксис метода

[модификаторы] возвращаемый_тип имя_метода([параметры]) {
    // тело метода
    return значение; // если не void
}

Пример

public static int sum(int a, int b) {
    return a + b;
}

2. Основные компоненты

  • Модификаторы: public, private, static, final
  • Возвращаемый тип: void, int, String и т.д.
  • Параметры: входные данные

3. Перегрузка методов (Overloading)

Методы с одинаковым именем, но разными параметрами.

public static int sum(int a, int b) { return a + b; }
    public static double sum(double a, double b) { return a + b; }
    public static int sum(int a, int b, int c) { return a + b + c; }

4. Рекурсия

Метод вызывает сам себя. Обязательно нужен базовый случай.

public static int factorial(int n) {
    if (n == 0) return 1;           // базовый случай
    return n * factorial(n - 1);    // рекурсивный вызов
}

5. Varargs (переменное количество аргументов)

public static int sum(int... numbers) {
    int total = 0;
    for (int num : numbers) {
        total += num;
    }
    return total;
}

// Вызов:
sum(1, 2, 3);        // → 6
sum(10, 20, 30, 40); // → 100

6. Практические примеры

// Максимум из двух чисел
public static int max(int a, int b) {
    return (a > b) ? a : b;
}

// Приветствие
public static String hello(String name) {
    return "Привет, " + name + "!";
}

Советы по написанию методов

  • Давать осмысленные имена (глаголы): calculateTotal, getUserName
  • Один метод — одна задача
  • Не больше 5–7 параметров
  • Использовать private для вспомогательных методов
