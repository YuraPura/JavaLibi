    1. Live Templates (Шаблоны кода)

    Предопределённые фрагменты для быстрой вставки.
    Активация: ввести аббревиатуру → Tab.

    Аббревиатура Результат
    psvmpublic static void main(String[] args) { }
    soutSystem.out.println();
    foriЦикл for с индексом для массивов
    iterЦикл for-each для коллекций
    soutvПечать переменной: System.out.println("var: " + var);

    Пример:

    // Введите iter → Tab
for (String item : list) {
    // Курсор здесь
}

    2. Постфиксные сокращения

    Оборачивают существующий код в конструкции.
    Активация: [выражение].[постфикс] → Tab.

    ПостфиксРезультат
    .ifnif (var == null) { }
    .nnif (var != null) { } (аналог .notnull)
    .varСоздаёт переменную: Type name = value;
    .forЦикл for для массива/коллекции

    Пример:

    user.ifn → 
if (user == null) {
    // Курсор здесь
}

    4. Настройка и советы

    Настройка шаблонов

    1. File → Settings → Editor → Live Templates (Win/Linux)
      IntelliJ IDEA → Preferences → Editor → Live Templates (macOS).
    2. Создавайте кастомные шаблоны для:
      — Логирования: logd → log.debug("Message: {}", var);
      — Тестов: testm → генерация метода теста.

    Лайфхаки

    • Быстрый просмотр реализации: Наведите курсор → Ctrl + Shift + I / Cmd + Shift + I
    • Поиск по действиям: Ctrl + Shift + A / Cmd + Shift + A
    • Мультикурсор: Alt + клик
    Пример использования:
    1. psvm → Tab → готовый main
    2. new ArrayList<>().var → Tab → List<String> list = new ArrayList<>();
    3. Выделите list → .for → Tab → for-each
