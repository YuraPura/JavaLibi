Массивы в Java

1. Массивы в Java

1.1 Что такое массив?

Массив — структура данных, хранящая несколько элементов одного типа. Размер фиксированный.

1.2 Объявление массива

// Вариант 1
    int[] arr = new int[5];

    // Вариант 2
    int arr[] = new int[5];

1.3 Инициализация массива

int[] arr = {1, 2, 3, 4, 5};

1.4 Доступ к элементам

int x = arr[0];  // первый элемент
    arr[1] = 10;     // изменение второго элемента

1.5 Размер массива

int size = arr.length;

1.6 Перебор массива

// Обычный for
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

// For-each
for (int elem : arr) {
    System.out.println(elem);
}

2. Многомерные массивы

2.1 Объявление

int[][] matrix = new int[3][4]; // 3 строки × 4 столбца

2.2 Инициализация

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

2.3 Доступ к элементам

int element = matrix[1][2]; // строка 1, колонка 2
    matrix[0][0] = 10;

2.4 Перебор

for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

2.5 Нерегулярные массивы

int[][] irregular = {
    {1, 2, 3},
    {4, 5},
    {6, 7, 8, 9}
};

3. Особенности массивов в Java

  • Фиксированный размер (нельзя изменить после создания)
  • Нулевая инициализация: int[] → 0, объекты → null
  • Могут быть примитивных и ссылочных типов

4. Заключение

Массивы — мощный инструмент для работы с данными. Эффективный доступ по индексу, но ограничение — фиксированный размер. Для динамических коллекций используйте ArrayList и другие коллекции.
