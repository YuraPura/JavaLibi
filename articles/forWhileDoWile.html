Циклы в Java

Циклы используются для многократного выполнения блока кода, пока выполняется указанное условие.

1. Цикл for

Используется, когда количество итераций известно заранее.

Синтаксис:

for (инициализация; условие; обновление) {
    // Код, выполняемый на каждой итерации
}

Пример:

for (int i = 1; i <= 5; i++) {
    System.out.println("Итерация: " + i);
}

2. Цикл while

Используется, когда количество итераций заранее неизвестно.

Синтаксис:

while (условие) {
    // Код выполняется, пока условие истинно
}

Пример:

int i = 1;
while (i <= 5) {
    System.out.println("Итерация: " + i);
    i++;
}

3. Цикл do-while

Выполняет код хотя бы один раз, даже если условие ложно.

Синтаксис:

do {
    // Код выполняется хотя бы один раз
} while (условие);

Пример:

int i = 1;
do {
    System.out.println("Итерация: " + i);
    i++;
} while (i <= 5);

4. Операторы break и continue

  • break – завершает выполнение цикла.
  • continue – пропускает текущую итерацию и переходит к следующей.

Пример использования break:

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // Остановит цикл на 5
    }
    System.out.println(i);
}

Пример использования continue:

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // Пропустит число 3
    }
    System.out.println(i);
}

5. Вложенные циклы

Циклы могут находиться внутри других циклов.

Пример:

for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.println("i = " + i + ", j = " + j);
    }
}
