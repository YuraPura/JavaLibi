Основные типы данных в Java

Целочисленные типы данных

byte

  • Диапазон: от -128 до 127
  • Память: занимает 1 байт

short

  • Диапазон: от -32 768 до 32 767
  • Память: занимает 2 байта

int (самый популярный тип данных)

  • Диапазон: от -2 147 483 648 до 2 147 483 647
  • Память: занимает 4 байта

long

  • Диапазон: от -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807
  • Память: занимает 8 байт
  • Примечание: для литерала добавляют L
long bigNumber = 10000000000L;

Типы данных для дробных чисел

float

  • Память: занимает 4 байта
  • Особенность: обязательно добавлять f в конце
float num = 1.01f;

double

  • Память: занимает 8 байт
  • Особенность: позволяет хранить большое количество цифр
double preciseNumber = 3.141592653589793;

Типы данных для строк

char

Описание: хранит один символ в одинарных кавычках

char letter = 'm';

String

Описание: хранит строку символов в двойных кавычках

String greeting = "Hello, Java!";

Логический тип данных

boolean

Описание: хранит только true или false

boolean isJavaFun = true;
