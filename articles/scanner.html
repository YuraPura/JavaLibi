Получение данных от пользователя в Java

Импорт класса Scanner

import java.util.Scanner;

Создание объекта Scanner

Scanner scan = new Scanner(System.in);
  • scan — имя переменной (можно любое)
  • System.in — ввод с клавиатуры

Чтение данных от пользователя

Считывание строки

System.out.print("Введите ваше имя: ");
    String username = scan.nextLine();
    System.out.println("Привет, " + username);

nextLine() — читает всю строку до Enter

Считывание чисел

int num1 = scan.nextInt();     // целое число
    byte num2 = scan.nextByte();
    float flo = scan.nextFloat();
    double d = scan.nextDouble();

Считывание логического значения

boolean b = scan.nextBoolean(); // true или false

Очистка буфера Scanner

Проблема: После nextInt(), nextFloat() и т.д. в буфере остаётся символ перевода строки \n.
Из-за этого следующий nextLine() сразу «съедает» пустую строку.

Решение — очистка буфера

System.out.print("Введите ваш возраст: ");
    int age = scan.nextInt();
    scan.nextLine(); // ← ОЧИЩАЕМ БУФЕР!

    System.out.print("Введите ваше имя: ");
    String name = scan.nextLine();

    System.out.println("Возраст: " + age + ", Имя: " + name);

ВАЖНО!

После использования Scanner желательно закрыть его: 

scan.close();
